# Paladugu Ganesh Naidu - Portfolio

A responsive single-page portfolio built with HTML, CSS, and JavaScript.

## Included Sections

- Hero section with looping typewriter animation for the name.
- Scroll indicator that fades out as you scroll.
- Reusable glass-morphism cards with hover lift and blue border highlight.
- Skills icon grid (2 columns mobile, 4 columns desktop) with tooltip hints.
- Featured projects grid with four cards and tech icons.
- Interactive hover-to-reveal highlight cards.
- Education timeline, achievements/certifications, and contact CTA.

## Project Structure

- index.html
- css/style.css
- js/main.js

## Run Locally

1. Open the folder in VS Code.
2. Open index.html directly in your browser, or run with Live Server.
3. No build tools or package installation required.

## Quick Content Updates

- Update hero typing text in js/main.js (fullName constant).
- Update section content in index.html.
- Update visual theme variables in css/style.css under :root.

## Deployment

You can deploy this directly on:

- GitHub Pages
- Netlify
- Vercel (static site)

For GitHub Pages:

1. Push this folder to a GitHub repository.
2. In repository settings, enable Pages.
3. Set branch to main and folder to root.
4. Save, then open the generated Pages URL.
